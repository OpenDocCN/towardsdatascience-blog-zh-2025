# 关于《数据科学之路》

> [原文链接](https://towardsdatascience.com/about-towards-data-science-d691af11cc2f/)

《数据科学之路》始于 2016 年，作为一个由社区驱动的中心，数据科学和机器学习从业者可以与同行分享他们的知识和想法——无论他们的位置或背景如何。这个基本目标至今依然不变。

几年来，我们已经发展成为一个拥有数百位活跃作者、一支确保我们发表文章质量的编辑团队，以及在我们主要网站、通讯和社交媒体渠道上拥有数百万读者和粉丝的出版物。

我们非常高兴成为数据科学、机器学习、人工智能和相邻主题学习的领先中心，尤其是在这些领域的发展影响到了世界上越来越多的个人、社区和社会。我们同样自豪地成为了一个欢迎和包容的社区，在这里，来自世界各地的作者和读者，代表广泛的观点和经验水平，汇聚在一起，共同拥有相互尊重的感觉，并专注于专业和智力成长。

在 2024 年，洞察媒体集团[收购了《数据科学之路》出版物](https://towardsdatascience.com/community-announcement-insight-media-group-llc-acquires-towards-data-science-publication-fc1adcb3e378/)。第二年，我们[推出了我们自己的自托管网站和贡献者平台](https://towardsdatascience.com/towards-data-science-is-launching-as-an-independent-publication/)，使我们能够托管并支付贡献者的费用。

自从我们的早期以来，我们已经以许多方式发展，我们确信我们将继续与我们的社区一起成长。永远不会改变的是，我们分享我们所能找到的关于我们涵盖主题的最好写作的使命。感谢您加入我们的旅程。

对于那些希望更积极地参与我们出版物的人来说，我们总是愿意考虑那些有见解和专业知识与我们核心关注领域（数据科学、机器学习、人工智能和相关编程主题）分享的新作者的作品。
