+   **[LLM 时代评估的范式转变](https://towardsdatascience.com/paradigm-shifts-of-eval-in-the-age-of-llm-7afd58e55b29)** 是时候重新评估我们对待评估的方式了吗？Lili Jiang 认为是时候了：“我逐渐认识到，LLM 需要我们在评估思考方式上做出一些微妙、概念上简单但重要的改变。”她的最新文章提供了对新的范式可能是什么样子的高级见解。

+   **[LLM 准确性的下一个前沿](https://towardsdatascience.com/the-next-frontier-in-llm-accuracy-cb2491a740d4)** 在 LLM 优化主题上保持紧密联系，Mariya Mansurova 的新深度分析详细介绍了我们可以用来提高模型准确性的几种方法，并聚焦于高级微调技术。

+   **[如何构建一个图 RAG 应用](https://towardsdatascience.com/how-to-build-a-graph-rag-app-b323fc33ba06)** 准备好卷起袖子深入代码了吗？Steve Hedden 的关于创建第一个图 RAG 应用的全面教程是任何对这一热门话题感兴趣但需要指导和背景以确保他们正确起步的人的理想选择。

+   **[使用 Hugging Face Code Agents 的多代理 RAG](https://towardsdatascience.com/multi-agentic-rag-with-hugging-face-code-agents-005822122930)** 基于代理的系统去年获得了巨大的势头（和关注），而且看起来在 2025 年这种情况不会改变。好奇想了解更多关于它们的信息？Gabriele Sgroi, PhD 的耐心、逐步指南可能很长，但它仍然易于理解且清晰，概述了利用一个“小型”LLM 来驱动多代理系统并产生良好结果的过程——即使在消费级硬件上也能做到。

+   **[使用 Darts 进行需求预测：教程](https://towardsdatascience.com/demand-forecasting-with-darts-a-tutorial-480ba5c24377)** 虽然 LLMs（大型语言模型）目前吸引了我们大量的关注，但以业务为导向的工作流程仍然是行业数据科学家的工作重点。Sandra E.G. 的首篇 TDS 文章提供了一次扎实、实用的入门介绍，介绍了这样一个基本任务：在零售销售背景下进行需求预测。

+   **[使用 Ray 轻松实现分布式并行计算](https://towardsdatascience.com/distributed-parallel-computing-made-easy-with-ray-f9043bb121b9)** 对于数据和 ML（机器学习）从业者来说，尝试新的工具和框架至关重要，因为看似微小的改进可以积累成重大的成本和效率效益。Betty LD 带我们了解她最近对 AI 重点的 Ray 库进行分布式数据处理的研究，并通过可扩展的离线批量推理用例展示了其功能。
