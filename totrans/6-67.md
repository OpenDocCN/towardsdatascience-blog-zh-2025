## **卡方检验**

卡方检验是一种统计方法，帮助我们确定观察到的分类变量之间的差异（或相似性）是偶然发生的，还是反映了真实而有意义的关系。这使得它成为假设检验中的必备工具，在数据科学和其他领域得到广泛应用。

*如果您还记得，分类变量是数据中的一种变量类型，它代表的是组或类别，而不是数值。*

为了确定某物是否为分类变量，我通常问自己："这有没有有限个不同的组？" 例如，如果我们正在分析调查中的 **"首选饮料选择"**，选项可能是咖啡、茶、水、苏打水或果汁这样的类别。由于我可以清楚地列出这些不同的组，我知道它是一个分类变量。

### **Chi-Squared 检验的类型**

Chi-Squared 检验主要有三种类型，每种都服务于特定的目的：

+   **独立性 Chi-Squared 检验** 确定在 **单个总体** 中两个分类变量之间是否存在显著关联。

+   **同质性 Chi-Squared 检验** 比较一个分类变量在 **两个或更多不同总体** 或 **群体** 中的分布。

+   **Chi-Squared 好拟合度检验** 确定单个分类变量的观察分布是否与理论或预期分布相匹配。

如果这看起来很困惑，不要担心！我会用详细的例子来分解每种测试类型，使其更清晰。

### **关键概念**

Chi-Squared 检验是一种假设检验方法

+   **零假设 (H₀)：** 假设不存在显著的关系或差异。

+   **备择假设 (Hₐ)：** 假设存在显著的关系或差异。

+   **右尾检验：** Chi-Squared 检验总是右尾检验，因为检验统计量 (χ²) 基于平方差，平方差总是正的。χ² 的大值表明与零假设有显著偏差，导致其被拒绝。

+   **列联表：** 通常在独立性 Chi-Squared 检验中，你会看到某种表格，显示每个分类变量的数据（如下所示）。它总结了数据，以了解变量之间的关系。

## 分析意甲联赛中的比赛风格：独立性 Chi-Squared 检验

想象一下，你正在为 AC 米兰的主教练担任**数据科学家**。你和主教练被球队雇佣，因为新赛季即将到来（2025–2026）。AC 米兰以其强大的防守足球风格而闻名。然而，这种方法在近年来并不成功。

随着新赛季的临近，教练渴望了解某些**比赛风格**——例如平衡、控球或反击——是否与意甲球队的**更好的比赛结果**（胜利、平局或失败）相关。

为了测试这一点，你将分析整个过去意甲赛季（2024–2025）的比赛结果。这些数据代表了一个**单一总体**：该赛季在意甲进行的所有比赛。你正在检查两个分类变量之间的关系：

+   **比赛风格**：平衡、控球、反击。

+   **比赛结果**：胜利，平局，失败。

为了简化，我们假设的意甲分析数据集总结在下表中。

![我们意甲示例的观察数据](img/b67824fbcd49e9fd4646d0aa30cb2be3.png)

我们意甲示例的观察数据

**读者注意**：虽然现实世界的比赛结果受多种因素影响——如球队深度、球员可用性和伤病——但我们将仅关注比赛风格进行分析！

### 定义假设

我们如何使用统计测试来确定某些比赛风格是否会影响比赛结果？嗯……记住，这是一个假设测试，所以让我们定义零假设和备择假设！

+   **零假设 (Hₒ)**：比赛风格与比赛结果无关。 (比赛风格与结果之间不存在关系。)

+   **备择假设 (Hₐ)**：比赛风格与比赛结果相关。 (两个变量之间存在关系。)

为了测试这些假设，我们将使用**卡方检验**来评估观察到的比赛结果差异是否具有统计学意义！

### 我们从我们的数据中期待什么？

在我们开始进行测试之前……让我们退一步，考虑卡方检验独立性试图实现的目标。在本质上，可以说这个测试询问的是：

+   比赛风格之间的比赛结果差异仅仅是随机的吗？

+   或者这些差异反映了比赛风格与结果之间的有意义关系？

> 现在，让我们从直观的角度来考虑这个问题。

如果比赛风格对比赛结果没有影响，那么每种风格的**胜利、平局和失败的数量应该与数据中的整体比例相一致**。例如，如果 50%的所有比赛都以胜利结束，我们预计每种比赛风格将有大约 50%的胜利！

如果我们要计算每种比赛风格与比赛结果之间相关性的预期数据……它可以根据行和列的总数简单计算。

这是根据我们的数据，我们预期表中每个单元格的值应该是多少

例如，平衡玩法的预期胜利次数如下所示。我们可以对每个单元格都做同样的处理，以得到预期数据！

![平衡玩法的预期胜利次数](img/15b65f5a759074d8e46112019b21b9b9.png)

### 这有什么意义？

好的，我们直观地讨论了我们预期数据应该是什么样子。但是，为什么我们关心**预期数据**而不是只看**观测数据**呢？

![预期数据](img/905e3f6b5b88ba48fc7e52b90df040f3.png)

预期数据

嗯……预期数据对我们卡方检验至关重要，因为它作为比较的基础。通过计算预期值，我们本质上是在说：

+   *"如果玩法对比赛结果没有影响（即不存在关系），数据应该看起来是这样的。"*

当我们比较观测数据和预期数据时，任何显著的偏差都可能表明玩法和比赛结果有关联。

![观测数据](img/b67824fbcd49e9fd4646d0aa30cb2be3.png)

观测数据

这是有意义的，对吧？如果确实没有关系，观测数据应该与预期数据紧密一致。另一方面，如果观测数据与预期数据明显不同，这表明玩法和比赛结果之间可能存在某种关系。

### 理解卡方检验

这种比较是卡方检验的基础，帮助我们确定差异是否具有统计学意义或只是随机机会。

在这个测试中，关键指标是卡方统计量（χ²），它衡量观测数据与预期数据之间的偏差程度，假设两个变量是独立的。计算卡方统计量的公式是：

![我们的卡方统计量！](img/3950f28031512fe5e9d487fe97ae6382.png)

我们的卡方统计量！

+   **O** 是列联表中每个单元格的观测频率。

+   **E** 是列联表中每个单元格的预期频率。

对于表格中的每个单元格，你计算这个值，然后将它们全部加起来得到总的卡方统计量！

## 使用 Python 进行测试

现在，你可以使用 Python 来计算卡方统计量，并确定观测到的差异是否具有统计学意义！我们将显著性水平（α）设置为 0.05。

```py
import numpy as np
from scipy.stats import chi2_contingency

# Observed data
observed = np.array([
    [80, 50, 80],  # Balanced
    [90, 20, 10],  # Possession
    [30, 30, 20]   # Counter-Attack
])

# Perform Chi-Squared Test
chi2, p_value, dof, expected = chi2_contingency(observed)

# Results
print("Chi-Squared Statistic:", chi2) # 72.34
print("P-Value:", p_value) # 0.00001
print("Degrees of Freedom:", dof) # 4
print("Expected Frequencies:n", expected) # Table for expected values

# Significance level
alpha = 0.05
```

通过查看我们的数据，我们得到了 p 值为 **0.00001**，这小于显著性水平（α=0.05）。这意味着什么？这意味着我们**拒绝零假设**。这表明在意大利足球甲级联赛中，玩法和比赛结果之间存在统计学上的显著关系。

因此，我们可以告诉教练，我们可以探索采用**控球式风格**，因为它根据分析显示与更好的比赛结果（例如，更多胜利）有更强的关联！

## 点球成功率：卡方检验的同质性检验

对于一个数据科学项目，我分析了来自各种高知名度足球锦标赛的点球成功率数据——分为**进球**、**被扑救**或**失误**——包括**欧洲冠军联赛**、**FIFA 世界杯**和**南美杯**。

本项目的目标是调查在这些锦标赛中点球的成功率是否有所不同，或者保持一致，考虑到每个事件中不同的压力和竞争水平。

> 有趣吧？

这种问题非常适合用**卡方检验的同质性检验**来解决！记住，在这个卡方检验的变体中，你是在比较**分类变量的分布**（点球成功率）是否在多个组或群体**（不同的足球锦标赛）**中**相同**。

下面是数据，自己尝试一下卡方检验的同质性检验吧！

![点球成功率观察数据](img/bb46de2eea3f3ed36d7a3fd2d9729437.png)

点球成功率观察数据

### 答案

```py
import numpy as np
from scipy.stats import chi2_contingency

# Observed data
observed = np.array([
    [40, 35, 30],  # Goal
    [20, 25, 15],  # Saved
    [10, 15, 20]   # Missed
])

# Perform the Chi-Squared Test
chi2, p_value, dof, expected = chi2_contingency(observed)

# Print results
print("Chi-Squared Statistic:", chi2)
print("P-Value:", p_value)
print("Degrees of Freedom:", dof)
print("Expected Frequencies:n", expected)

# Chi-Squared Statistic: 7.688
# P-Value: 0.053
# Degrees of Freedom: 4
# Expected Frequencies:
 [[35.0 37.5 32.5]
  [20.0 21.43 18.57]
  [15.0 16.07 13.93]]
# Fail to reject the null hypothesis: No significant difference in success rates across tournaments.
```

## 足球比赛中的进球数：卡方检验的拟合优度检验

想象一下，你想测试足球比赛中进球数的分布是否与**[泊松分布](https://medium.com/p/ff8a6ddeb4a1)**相匹配。如果你还记得，泊松分布通常用于模拟在固定间隔内发生的事件数量（如进球），例如单场足球比赛。

> [**概率分布：泊松分布与二项分布**](https://towardsdatascience.com/probability-distributions-poisson-vs-binomial-distribution-ff8a6ddeb4a1)

根据历史联赛数据，你假设进球数的以下概率分布：

![观察数据和理论概率](img/8238e855a74703f063ab6e6c8b95e65c.png)

观察数据和理论概率

### 观察数据和理论概率

这种问题非常适合用**卡方拟合优度检验**来解决！记住，我们使用这种变体来确定一个**单个分类变量（在这种情况下，进球数）是否遵循特定的理论或期望分布（例如，泊松分布）**。

通过比较观察数据与理论概率，我们可以确定这两个分布是否一致或存在显著差异。

这意味着与我们的先前版本卡方检验不同，我们的假设将试图查看分类变量的分布是否与我们的理论分布相匹配！

+   **零假设 (H₀)**: 观察到的进球分布与理论泊松分布相匹配。

+   **备择假设 (Hₐ)**: 观察到的进球分布不匹配理论泊松分布。

由于计算预期数据可能有点令人困惑，我将为你提供它！自己尝试使用卡方拟合优度检验！

![预期数据](img/e0789202a67e5dc128f0948952c766e2.png)

预期数据

### 答案

```py
import numpy as np
from scipy.stats import chisquare

# Observed data
observed = np.array([50, 70, 60, 20])

# Expected data based on theoretical probabilities
expected = np.array([50, 70, 50, 30])

# Perform the Chi-Squared Goodness-of-Fit Test
chi2, p_value = chisquare(f_obs=observed, f_exp=expected)

# Print results
print("Chi-Squared Statistic:", chi2)
print("P-Value:", p_value)

# Chi-Squared Statistic: 1.8
# P-Value: 0.614
# Fail to reject the null hypothesis: The observed distribution matches the expected distribution.
```

## 摘要

以足球作为案例研究，我们得以看到**卡方独立性检验**如何揭示变量之间的关系，例如比赛风格和比赛结果，使教练能够更有效地制定策略。

同样，**卡方同质性检验**有助于比较不同组之间的分布，例如不同锦标赛的罚球成功率。

那么**卡方拟合优度检验**呢？它使我们能够评估现实世界数据，如比赛中进球数，是否与理论模型（如泊松分布）一致，从而让我们更好地了解游戏模式和进一步分析的可能问题。
