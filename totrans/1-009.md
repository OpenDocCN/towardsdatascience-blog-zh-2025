## 纯度度量作为分割的标准

在决策树回归器中，我们已经知道：

+   对于一个区域的预测是目标变量的**平均值**。

+   分割的质量是通过**MSE**来衡量的。

在决策树分类器中：

+   对于一个区域的预测是该区域的**多数类**。

+   分割的质量是通过一个**纯度度量**来衡量的：**Gini纯度**或**熵**。

这两种在教科书中都是标准的，并且都在scikit-learn中可用。默认使用Gini。

但是，这个纯度度量实际上是什么？

如果你看看**基尼**和**熵**的曲线，它们都以相同的方式表现：

+   当节点是**纯的**（所有样本具有相同的类别）时，它们是**0**。

+   当类别均匀混合时（50/50），它们达到**最大**。

+   曲线是**平滑**、**对称**的，并随着无序的增加而增加。

这是任何**不纯度度量**的基本属性：

> 当组干净时，不纯度低，当组混合时，不纯度高。

+   我们列出所有可能的分割，

+   我们在每一侧计算不纯度，

+   我们取加权平均值，

+   我们选择具有最低不纯度的分割。

## 真的不纯度度量之间有多大的不同？

现在，我们总是提到基尼或熵作为标准，但**它们真的不同吗**？当查看数学公式时，有些人可能会说

**答案不是很多。**

理论上，在几乎所有实际情况下：

+   基尼和熵**选择相同的分割**

+   树结构几乎是**相同**的

+   预测是**相同的**

为什么？

因为它们的曲线看起来非常相似。

它们在 50% 混合时达到峰值，在纯净度时降至零。

唯一的区别是曲线的**形状**：

+   **基尼**是一个**二次**函数。它对误分类的惩罚更线性。

+   **熵**是一个**对数**函数，因此它在 0.5 附近对不确定性的惩罚要强一些。

但实际上差异很小，你可以在 Excel 中完成它！

### 其他不纯度度量？

另一个自然的问题是：是否可以发明/使用其他度量？

是的，你可以发明你自己的函数，只要：

+   当节点纯净时，它是**0**

+   当类别混合时，它是**最大**的

+   它在“无序”中是**平滑**且严格递增的

例如：不纯度 = 4*p0*p1

这是一种有效的不纯度度量。实际上，当只有两个类别时，它等于**基尼**乘以一个常数。

## Excel中的练习

### 使用其他参数和特征的测试

一旦你建立了第一个分割，你就可以扩展你的文件：

+   尝试使用**熵**而不是基尼

+   尝试添加**分类特征**

+   尝试建立**下一个分割**

+   尝试更改**最大深度**并观察欠拟合和过拟合

+   尝试为预测创建一个混淆矩阵

这些简单的测试已经让你对真实决策树的行为有了很好的直观感受。

### 为泰坦尼克号生存数据集实现规则

一个自然的后续练习是重新创建著名的**[泰坦尼克号生存数据集](https://www.kaggle.com/datasets/yasserh/titanic-dataset)**（**CC0 / 公共领域**）的决策规则。

首先，我们可以只从两个特征开始：**性别**和**年龄**。

在Excel中实现规则既长又有点繁琐，但这正是重点：它让你真正意识到决策规则的真实样子。

它们不过是一系列反复出现的**IF / ELSE**语句。

这就是决策树的真正本质：简单的规则，层层叠加。
