<mdspan datatext="el1756250453754" class="mdspan-comment">进入机器学习领域很困难。</mdspan>

在这个过程中，我使用了许多课程、书籍和资源，它们帮助了我，但说实话，许多我在事后不会选择。

因此，我想回顾一下我为了在机器学习领域找工作所学的所有东西，然后我会告诉你哪些领域是真正值得的，哪些不是。

让我们开始吧！

## 大学学位 / 数学

我非常幸运，在我还是青少年的时候决定学习物理学硕士学位。

是的，你现在可能正在翻白眼。

> “这个人说他没有计算机科学背景，但获得了物理学硕士学位，见鬼。”

我不能否认这确实给了我优势。然而，许多STEM毕业生仍然在机器学习领域找工作时遇到困难。我甚至亲自与他们合作过。

仅拥有STEM学科的硕士学位远不能保证你能够轻松找到工作。

你需要学习的东西还有很多，这些通常不是大多数课程中教授的内容。

就这样，我在学位期间学到的与我现在机器学习工程师工作相关的主要是数学技能。

我学习了微积分和线性代数，达到了非常高的水平，比实际需要的还要多，统计学则达到了相当的标准。即便如此，我后来还是不得不复习我的统计学知识。

我的学位也是我第一次写代码。

真的，我的第一天早上9点，我就有一个Fortran的计算机实验室教程。[**Fortran**](https://en.wikipedia.org/wiki/Fortran)。

对于那些不熟悉的人来说，Fortran是20世纪50年代发明的最古老的“高级”编程语言。然而，2017年我们还在学习它。

Fortran几乎不适合初学者，它立刻让我对编程失去了兴趣。如果那时的我知道我今天会做什么就好了！

虽然我不喜欢Fortran，但它教会了我如何使用代码思考和解决问题，这在长远来看是值得的。

## Python

因为我很讨厌Fortran，所以我主动避免任何带有编程方面的模块。

然而，在2020年，在我大三的时候，我的YouTube主页上推荐了一个视频。

[***AlphaGo — The Movie***](https://www.youtube.com/watch?v=WXuK6gekU1Y&t=1539s)

对于那些不知道的人来说，这是一部关于DeepMind的AI AlphaGo击败世界最佳围棋选手的纪录片。大多数人认为AI永远不可能擅长围棋，更不用说击败世界冠军了。

在观看视频后，我开始阅读关于AI如何工作的内容，包括神经网络、强化学习和深度学习。

从那时起，我决心成为一名数据科学家，我知道我必须学习Python才能实现这一目标。

在晚上和周末，我会浏览几个Python课程和项目，我使用的是：

+   [**学习Python — 初学者完整课程[教程]**](https://www.youtube.com/watch?v=rfscVS0vtbw)

+   [**W3Schools Python教程**](https://www.w3schools.com/python/)

+   [**Tutorialspoint Python教程**](https://www.tutorialspoint.com/python/index.htm)

更不用说我在ChatGPT之前进行的无数Google搜索和StackOverflow帖子了。

我还在[HackerRank](https://www.hackerrank.com/)上练习Python技能，并为乐趣以及大学课程构建了基本项目。

## SQL

在学习Python之后，我花了一个多月的时间学习SQL，同时申请初级和研究生数据科学职位。

SQL比许多其他语言更容易学习，因为它更小，基础涵盖了你想做的几乎所有事情。

我用于SQL的课程和资源包括：

+   [**完整的SQL入门课程：从零到英雄**](https://www.udemy.com/course/the-complete-sql-bootcamp/)

+   [**W3Schools SQL教程**](https://www.w3schools.com/python/)

+   [**Tutorialspoint SQL教程**](https://www.tutorialspoint.com/python/index.htm)

再次，我使用[HackerRank](https://www.hackerrank.com/)练习SQL面试题。

这只是我学习旅程的一部分，我在工作中获得了大部分高级SQL技能。

## 统计学

在我的学习旅程的这个阶段，我在一家保险公司找到了我的第一份数据科学家工作，在那里我密切地与精算师合作。

对于那些不知道精算师是做什么的人来说，[**维基百科**](https://en.wikipedia.org/wiki/Actuary) 这样描述他们：

> 精算师是具有高级数学技能的专业人士，他们处理风险和不确定性的测量和管理。

尽管我在之前学习过统计学，但保险公司要求的水平相对较高，尤其是在与精算师合作时，因为他们是这个领域的专家。

为了提升我的统计学水平，我学习了[**CS1**](https://actuaries.org.uk/qualify/curriculum/actuarial-statistics/)（统计学）精算师考试。尽管我实际上没有参加考试，但我回顾并学习了所有内容。

课程大纲几乎涵盖了你在整个职业生涯中作为数据科学家或机器学习工程师可能用到的所有统计学知识。

书籍[**《数据科学家实用统计学**》](https://amzn.to/45V0Bdv) **(推广链接**)作为参考文本来刷新我的知识，我学习了[**《贝叶斯思维**》](https://amzn.to/4oRuLa5) **(推广链接**)教科书来学习贝叶斯统计学。

重要的是要注意，我不仅仅参加了课程和阅读了书籍；我在Medium上记录了几乎所有我在学习过程中学到的东西。

[**通用统计学**](https://medium.com/@egorhowell/list/006e8c3cee05)

[**概率分布**](https://medium.com/@egorhowell/list/1421ad42a84b)

[**贝叶斯统计**](https://medium.com/@egorhowell/list/36268a8570be)

## 软件工程

当我打算从数据科学家转型为机器学习工程师时，我需要提高的关键领域是我的软件工程技能。

这是一个很大的领域，实际上，它是一个完整的职业，但我专注于基础知识。

我所参加的课程有：

+   [**Neetcode: 初学者算法和数据结构**](https://neetcode.io/courses/dsa-for-beginners/0) — 我使用这个资源来学习面试所需的DSA（数据结构与算法），同时也了解了编程中使用的根本算法。

+   [**Neetcode: 初学者系统设计**](https://neetcode.io/courses/system-design-for-beginners/0) — 与前面的资源类似，我利用这个资源来深入理解面试所需的系统设计，并帮助我开发更有效的规模化机器学习系统。

+   [**实用MLOps**](https://amzn.to/45Ap20S) **（推广链接）** — 一本很好的教科书，教你如何部署像[**Kubernetes**](https://kubernetes.io/)、[**Docker**](https://www.docker.com/)和一般的[**DevOps**](https://en.wikipedia.org/wiki/DevOps)这样的ML模型。

+   [**CS107 计算机组织与系统**](https://web.stanford.edu/class/archive/cs/cs107/cs107.1224/)**— **我在第一份工作中学习了这门课程，以更深入地了解计算机底层的工作原理，并学习一些C语言。

一个难以学习的研究领域是编写正确的生产代码。这是我唯一完全在工作中学习的东西，但你也可以通过创建自己的软件项目来获得这方面的经验。

* * *

如果这看起来很多，不要担心，因为这几乎相当于近5年每天持续学习的结果！

此外，正如我一开始所说的，事后看来，其中很多并不是必需的。以下是一些我肯定不会再次做的事情。

+   **精算学CS1**— 许多概念在实际中并不需要，数学细节可能过于详尽。我建议坚持使用[**数据科学家实用统计学**](https://amzn.to/45V0Bdv) **(联盟链接**)教科书。

+   **CS107 计算机组织与系统**— 实际上并没有在这里学到很多想法。

+   **统计学习要素**——对大多数人来说是一本过于详尽的教科书。

其余的确实值得，但我确实不需要所有那些资源。每个部分有一个好的就足够了。
