在许多健康调查的合并数据集中，其中一些可能没有所有三个变量（PSU、层和抽样权重），如果你要进行完全案例分析，你将只保留包含这三个变量数据的数据集。这可能会导致样本量大幅减少，因为我们需要排除例如只有 PSU 和抽样权重但没有层变量数据的数据集。

我的解决方案？独立分析每个数据集，利用它们拥有的任何变量（例如，只有 PSU 和抽样权重，没有层）。不幸的是，一次分析一个数据集将非常低效。然而，在超过七年的专业研究经验和数十篇使用健康调查的同行评审论文中，我遇到了这个问题，并想出了一套函数和步骤。

* * *

1.  如果 PSU、层和抽样权重中的缺失值数量与该数据集的样本量不同，则在 PSU、层和抽样权重中删除缺失观测值。换句话说，如果有一个包含 100 个观测值的数据集，其中 20 个在 PSU 中缺失数据，我将排除这 20 个观测值。相反，如果 PSU 中的缺失值数量为 100（即 PSU 完全为空），我将保持数据集不变。因为这表明 PSU 变量可能根本不可用，或者简单地没有包含在数据集中，就像我们下载的数据集可能那样。

1.  在每一项调查中，我将创建一个名为`type`的新变量，用于识别三个变量中是否完全缺失、两个完全缺失、三个完全缺失，以及所有可能的组合。例如，如果三个变量都存在，则`type` = 1；如果层和抽样权重缺失，则`type` = 2，依此类推。

1.  使用可用的任何变量（变量）来计算感兴趣的指标。这将利用之前步骤中创建的`type`变量中的信息。例如，如果`type`将此数据集标记为仅具有 PSU 和抽样权重，对于该特定数据集，我们将利用这两个变量来计算结果，而对于具有三个完整变量的其他数据集，结果将使用 PSU、层和抽样权重来计算。

诀窍是将每个数据集放入一个列表中，并应用函数到列表中的每个元素（即列表中的每个数据集或调查）。这样，如果需要，你也可以独立地处理每个数据集。

首先，我将解释每个函数及其功能。然后，我将提供一个使用这些函数的实际示例。

* * *

**A.** **处理缺失值**

当将这些函数应用于每个数据集时，如果缺失值的数量与数据集中观测值的总数不同，将排除缺失观测值。

例如，如果数据集有 100 人，其中 20 人缺失 PSU 变量，这 20 个观测值将被删除。相反，如果 100 人缺失 PSU 变量，我们将保持数据集不变。

这种方法假设，如果所有观测值对于 PSU 变量都有缺失数据，那是因为这是设计上的情况，或者这些信息没有与数据集一起提供。在任何情况下，我们几乎无法做什么。但我们仍然可以利用可用的任何信息。也就是说，如果 PSU 完全缺失，但数据集有层和抽样权重，我们可以在我们的分析中利用这些变量。

```py
countNA <- function(x)sum(is.na(x))

cleaning_svy <- function(df){
  if (countNA(df$psu) != nrow(df)){
    df <- df[which(!is.na(df$psu)), ]
  } 
  if (countNA(df$stratum) != nrow(df)){
    df <- df[which(!is.na(df$stratum)), ]
  }
  if (countNA(df$wstep2) != nrow(df)){
    df <- df[which(!is.na(df$wstep2)), ]
  }
  return(df)
}
```

* * *

**B.** **标记数据集**

我们排除了 PSU、层和抽样权重中任一变量缺失数据的少数观测值，同时保留了任一这些变量全部缺失值的数据集。

在这一步中，我们将创建一个新的变量，根据数据集包含的变量来标记每个数据集。例如，我们将确定数据集是否包含所有三个变量，只有一个（例如，仅 PSU），只有两个（例如，仅 PSU 和抽样权重），以及所有可能的成对组合（例如，PSU 和抽样权重但没有层）。

这个新变量，我们将命名为`type`，将用于稍后使用可用数据计算结果。例如，如果一个数据集被标记为包含所有三个变量，我们将使用这三个变量来计算结果；而如果一个数据集被标记为仅包含 PSU 和抽样权重，我们将仅使用这两个变量来计算结果。

```py
grouping_svy <- function(df){
  df$type <- ifelse(!is.na(df$psu) &amp; !is.na(df$stratum) &amp; !is.na(df$wstep2), 1,  # all three
             ifelse(is.na(df$psu) &amp; !is.na(df$stratum) &amp; !is.na(df$wstep2), 2,   # stratum &amp; weights
             ifelse(!is.na(df$psu) &amp; is.na(df$stratum) &amp; !is.na(df$wstep2), 3,   # psu &amp; weights
             ifelse(!is.na(df$psu) &amp; !is.na(df$stratum) &amp; is.na(df$wstep2), 4,   # psu &amp; stratum
             ifelse(!is.na(df$psu) &amp; is.na(df$stratum) &amp; is.na(df$wstep2), 5,    # psu
             ifelse(is.na(df$psu) &amp; !is.na(df$stratum) &amp; is.na(df$wstep2), 6,    # stratum
             ifelse(is.na(df$psu) &amp; is.na(df$stratum) &amp; !is.na(df$wstep2), 7,    # weights
             ifelse(is.na(df$psu) &amp; is.na(df$stratum) &amp; is.na(df$wstep2), 8, NA)))))))) # none
  return(df)
}
```

**C.** **计算结果**

我们在保持仅 PSU、层或抽样权重中任一变量缺失值的数据集的同时排除了缺失数据。我们还根据数据集包含的变量标记了数据集。现在，我们将计算感兴趣的成果。

在这个例子中，我们将计算一个名为`age`的变量的调查调整平均值。这是一个数值变量，我们将利用复杂的抽样设计来计算平均值和 95%置信区间。这意味着结果将代表潜在的总人口。例如，如果我们正在处理一个旨在提供代表一个国家结果的卫生调查，我们将计算该国的平均`age`。

首先，这个函数将应用于每个数据集，并查看变量`type`以了解哪些抽样设计变量可用，无论是 PSU、层还是抽样权重。其次，该函数将计算感兴趣的成果。在这种情况下，`age`的平均值。第三，结果将存储在数据框中，并将返回数据框。

您可以编辑第二步来计算您感兴趣的指标，无论是另一个数值变量的平均值还是分类变量的比例。在这个例子中，`results_svy_mean_age`函数将通过性别生成平均`age`。

```py
results_svy_mean_age <- function(df){
  options(survey.lonely.psu = "adjust", survey.adjust.domain.lonely = TRUE)
  if (unique(df$type == 1)) {
    dsub <- svydesign(id = ~psu, 
                      strata = ~stratum, 
                      weights = ~wstep3, 
                      data = df, 
                      nest = TRUE) } 
  if (unique(df$type == 2)) {
    dsub <- svydesign(id = ~1, 
                      strata = ~stratum, 
                      weights = ~wstep3, 
                      data = df, 
                      nest = TRUE)} 
  if (unique(df$type == 3)) {
    dsub <- svydesign(id = ~psu, 
                      strata = NULL, 
                      weights = ~wstep3, 
                      data = df, 
                      nest = TRUE)} 
  if (unique(df$type == 4)) {
    dsub <- svydesign(id = ~psu, 
                      strata = ~stratum, 
                      weights = NULL, 
                      data = df, 
                      nest = TRUE)} 
  if (unique(df$type == 5)) {
    dsub <- svydesign(id = ~psu, 
                      strata = NULL, 
                      weights = NULL, 
                      data = df, 
                      nest = TRUE)}
  if (unique(df$type == 6)) {
    dsub <- svydesign(id = ~1, 
                      strata = ~stratum, 
                      weights = NULL, 
                      data = df, 
                      nest = TRUE)}
  if (unique(df$type == 7)) {
    dsub <- svydesign(id = ~1, 
                      strata = NULL, 
                      weights = ~wstep3, 
                      data = df, 
                      nest = TRUE)}
  if (unique(df$type == 8)) {
    dsub <- svydesign(id = ~1, 
                      strata = NULL, 
                      weights = NULL, 
                      data = df, 
                      nest = TRUE)}

  age_m <- svyby(~age,
                              ~sex,
                              design = dsub, 
                              vartype = c("ci"),
                              svymean)

  age_m <- as.data.frame(age_m)

  return(age_m)
}
```

**D. 整合所有内容。**

假设您的包含许多卫生调查的合并数据集被称为`pooleddata`，并且有一个名为`study_id`的变量用于识别每个单独的调查。

```py
df <- list()
for (i in unique(pooleddata$study_id)){
  df[[i]] <- pooleddata[which(pooleddata$study_id == i),]
}
df <- lapply(df, cleaning_svy)
```

我们将创建一个空列表，并在列表中存储每个数据集作为列表的独立元素。我们将使用`study_id`来识别每个独特的调查。例如，如果我们的`pooleddata`是一个包含 10 个调查的池，我们的列表将有十个元素。

我们将应用`cleaning_svy`函数到列表的每个元素上。换句话说，应用到列表中存储的每个单独的数据集上。记住我们在步骤 A 中解释了`cleaning_svy`函数。

在步骤 A 结束时，我们将拥有包含我们单个调查的相同列表，但排除了抽样设计变量中的少数缺失观察值，同时保留了任何这些变量（PSU、层和抽样权重）中任何变量缺失的数据集。

我们现在将在列表中的每个数据集（即，每个调查）中创建`type`变量。因此，我们将拥有包含唯一调查的相同列表，每个调查都有一个名为`type`的新变量，该变量将标记哪些抽样设计变量可用。

df <- lapply(df, grouping_svy)

**总结**

如果你正在合并具有复杂抽样设计的多个健康调查，那么很可能其中一些调查没有用于计算考虑抽样设计的结果所需的三个变量（PSU、层和抽样权重）。你不必丢弃那些完全缺失这些变量的调查，你可以利用它们所拥有的变量（例如，PSU 和抽样权重，但没有层）独立分析每个调查。**我们只需三个简单步骤，就提供`R`函数来完成这项工作！**
