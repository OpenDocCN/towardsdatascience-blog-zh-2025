## 简介 – 半句真话就是整个谎言。

我不知道你是否曾经[见过福克斯新闻柱状图修剪“丑闻”](https://imgur.com/gallery/fox-news-is-great-math-3zmMKat)。福克斯决定调整 y 轴，使其看起来奥巴马医改随着时间的推移获得了很大的支持（不言而喻，时间类别应该是折线图，但这超出了本文的范围）。通过打破“始终从零开始”的“最佳实践”规则，该图表通过“半真话”来定制故事。

为了明确起见，这并不是一个政治声明，而是一个不良可视化实践声明。

为什么裁剪 y 轴是一种不良做法？当 y 轴被裁剪时，观众会失去对数据的上下文。例如，裁剪可能会使人们不清楚图表中的值是否接近“自然基线”（如零）或者值本身是相对较高或较低。

### 我们将在本博客中涵盖哪些内容？

在理解了你应该**始终**从 0 开始绘制柱状图之后，有些情况下，展示 y 轴的完整值域（从 0 到你想展示的任何值）实际上可能对讲述故事有害。那么在这些情况下我们能做什么呢？

+   **场景 1**：当柱状图**不**从 y=0 开始。

+   **场景 2**：偏差图 – 当你的基线不是 y=0。

+   **场景 3**：当从 y=0 开始会隐藏重要差异时。

## 场景 1\. 柱子可以不从 0 开始吗？

在柱状图中，柱子通常是从底部到顶部读取的，就像一座建筑。这就是为什么，在我们的脑海中——遵循建筑类比——我们期望柱状图从 0 开始。

柱子从不同于 0 的某个地方开始，但仍然容易一眼看懂的情况并不多见。我能想到的有 2 个：

+   水波图

+   柱状图引用地理度量等元素，其中基线可以是不同水平——海平面、平流层或地壳。

让我们看看一些例子。

### 示例 1\. 水波图

水波图在金融界非常常见。下面是一个我用 Plotly 创建的模拟水波图。

![模拟数据来源](img/078ece2d8320f09d58f0f2e7f65fa97e.png)

模拟数据来源

**水波图的特点是什么？**

1.  你可以看到有些柱子*不是*从 0 开始的。

1.  然而，柱子之间有一个连续的流动，这使得它非常类似于堆积柱状图。

1.  此外，对于每一步，都会绘制一条新的基线（用黑色线表示），这使每个柱子好像是从一个“0 型”基线开始的。

1.  最后，整个图表实际上是从 0 开始的。我们没有打破柱状图应该始终从 0 开始的规则。

### 如何创建此图的技巧

***如何在 Plotly 中构建水波图？***

```py
fig = go.Figure(
   go.Waterfall(
      orientation="v",
      measure=["relative", "relative", "relative", "relative", "total"],
      x=["Sales", "Consulting", "Purchases", "Other expenses", "Profit before tax"],
      textposition="outside",
      text=["+60", "+80", "-40", "-20", "80"],
      y=[60, 80, -40, -20, 0],
      connector={"line": {"color": "rgb(63, 63, 63)"}},
   )
)

fig.update_layout(
   title=dict(text="Profit and loss statement 2018",
              font=dict(family="Helvetica Neue", size=18),
              ),
   showlegend=False,
   margin=dict(t=100, pad=0),
   height = 500,
   width=600,
   font=dict(family="Helvetica Neue"),
)
```

### 示例 2\. 地理测量

现在想象一下，你是一位地理老师。你希望用以下问题挑战你的学生：（1）地球上最高的自然结构是什么？（2）地球上最高的山是什么？

## 场景 2：偏差图

想象一个场景，你想要比较类别，它们之间的差异很小，而且 y 轴的值很大。查看下面的数据框：

![数据来源：我们的世界数据](img/0e527618d73cab8750692f2178c002a1.png)

数据来源：[我们的世界数据](https://ourworldindata.org/gender-ratio#:~:text=Among%20first%2Dborn%20children%2C%20the,likely%20to%20have%20another%20child.)

### **第 1 次尝试：带有基准基线的简单柱状图**

我认为下面的图表是一个很好的起点。然而，由于柱子从 0 开始，值范围很大，柱子看起来像摩天大楼。我们想要展示的图表的主要点是高收入国家和低收入国家之间的差异，但有了这些摩天大楼，你实际上感觉不到中国和赞比亚之间的差异。

![图片](img/aa4fcb91bfd8fd3cb93b73449f849ca6.png)

### **第 2 次尝试：使用柱状图（即旗帜图）的偏差图**

旗帜图是一种修改过的柱状图。旗帜图有一个基线（可以是 0 或不同于 0），柱子位于“旗帜”的顶部（或下方）。旗帜图使得关注数据点之间的相对差异变得容易，特别适用于比较增量变化或显示与目标或基准的偏差。

![图片](img/3aabe94e82df7687fd0e68c10bb42007.png)

### 如何创建此图的技巧

+   **第 1 步**，绘制一条垂直基线。

```py
hist_avg = df['ratio_avg'].min().round(0)
fig.add_vline(x=0, 
   line_color="darkgrey", 
   line_width=3, 
   annotation_text=f'Historical average - {hist_avg}',
   annotation_position='bottom right'
)
```

+   **第 2 步**，创建一个偏差特征。

```py
return_df['deviation'] = return_df['ratio'] - hist_avg
```

+   **第 3 步**，绘制一个柱状图，其中 x 轴是“偏差”，但你显示的是实际比率。

```py
fig.add_trace(
        go.Bar(
            y=df['Entity_text'],
            x=df['deviation'],
            orientation='h',
            marker_color='darkblue',
            text=df['ratio'],
            textposition='outside',
            showlegend=False,
        )
    )
```

### **第 3 次尝试：使用点图的偏差图**

在我看来，前面的图表相当有效。然而，鉴于我们知道最佳实践表明柱状图应该始终从 0 开始，我们打破了这条规则。为了不打破这条规则，我们可以进行轻微调整，将偏差图从柱状图转换为使用点。

我们实际上在 *[Awesome Plotly with code series (Part 1): Alternatives to bar charts](https://medium.com/towards-data-science/awesome-plotly-with-code-series-part-1-alternatives-to-bar-charts-125502587690)* 中展示了一个“棒棒糖”图，所以这里的想法也是使用类似的东西。通过使用点图，而不是大色块柱子，我们的注意力集中在每个单独的点上，而不是整个柱子的长度。在我看来，这种偏差散点图要好得多。

![图片](img/c5725661c588a4dfab4ea098027d2190.png)

## 场景 3：隐藏在柱状图中的大数字背后。

在场景 2 中，我们已经看到了柱状图如何能隐藏在大量数字背后的微小变化。在场景 3 中，我们覆盖了同样的想法，但这次是变化发生在时间上的情况。

想象你是一名为 NHS 工作的高级顾问。你一直在收集关于 A&E 中等待 4 小时或更少（就像 4 小时是一个短的等待时间……）的患者的百分比数据。你知道 NHS 在 2018 年有一个 95% 的目标。因此，你想展示这个百分比指标的演变及其与基准的对比。以下是您收集的数据。

![数据来源：Nuffield Trust](img/1fded576677b2367d1a0cf3d6ad5f7b6.png)

数据来源：[Nuffield Trust](https://www.nuffieldtrust.org.uk/resource/a-e-waiting-times)

### **第一次尝试：如果你想对 NHS 表示友好，请使用柱状图。**

你首先使用柱状图。你不想违反不将柱状图从 0 开始的规则。总的来说，图表确实以某种方式显示了随时间的变化。它还显示你现在离 95% 的目标最远。

但它并没有传达出戏剧性的变化。因为它们从零开始，最高的起始点（70%）已经看起来很大了。换句话说，你是在展示真实的数据，但同时也是为了取悦顶层的 NHS 人士。

![](img/6c7bf15bccc89f492ffaa204de39bdd5.png)

### **第二次尝试：如果你想向 NHS 展示戏剧性的变化，请使用线形图。**

线形图和点状图并不是基于需要从 y=0 开始的需求来锚定的。正因为如此，我们可以放大到相关的值范围，以强调我们的故事。你能看到下面图表中的差异吗？

变化更加剧烈，营造出一种“停止下跌”的紧迫感。通过将最低百分比放在图表的最低部分，我们唤起了我们“触底”的感觉。在柱状图中，底线是 0，因此，我们离触底非常远。

![](img/99b010da81989efc4cede974b872af6f.png)

## 摘要

在这篇文章中，我们探讨了 y 轴操作对柱状图的影响。最佳实践规则是柱状图从零开始，以保持清晰和诚实。然而，如果你需要讲述一个特定的故事，你可能需要一个替代方案。例如，旗杆图或偏差图有助于关注基准或组之间的差异。此外，线形图不需要从 y=0 开始，这有助于关注趋势中的差异。

尽管如此，文章应该传达出……

> *柱状图应该**始终**从 0 开始。*
> 
> *如果你在考虑不从 0 开始，那么就切换图表类型。*
