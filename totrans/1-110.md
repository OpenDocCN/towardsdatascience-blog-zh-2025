# 为什么数据讲故事对商业和数据分析师来说很重要

> 原文：[https://towardsdatascience.com/why-storytelling-with-data-matters-for-business-and-data-analysts/](https://towardsdatascience.com/why-storytelling-with-data-matters-for-business-and-data-analysts/)

<mdspan datatext="el1762801975704" class="mdspan-comment">对于任何与数据打交道的企业</mdspan>，无论大小，真正为未来定调的是对数据的最佳解读。

作为在数据或商业分析领域工作的专业人士，你的角色不仅限于制作报告，因为你有能力将原始数据转化为能够激发行动的故事。我们都知道那种感觉，我们花费数小时创建仪表板、报告绩效指标或制作Excel电子表格，结果发现我们的利益相关者仍然难以将这些数字转化为有意义的决策。

**这就是讲故事的作用所在！**

借助正确的思维方式、工具和方法，你可以超越电子表格和图表，以构建更强大的商业案例、提高敏捷性并推动团队之间的协同。

太频繁了，分析师最终只是将数据作为Excel文件中的行和列或作为密集的报告来分享。五年前的我就是这样，信息都在那里，但很少能驱动任何行动，因为我的数据消费者们很难找到“所以呢”。现在，对比一下一张单一有力的总结幻灯片：一个清晰的标题、引人入胜的视觉元素，以及基于你分析的数据的强大、简洁的建议——同样的数据现在突然变得有价值、可操作和有说服力。

这就是用数字报告和用它们讲故事之间的区别！

从我与数据领导者并肩工作以及作为分析顾问近五年的经验来看，我了解到**用数据讲故事是将原始信息转化为洞察力的关键**。通过这篇文章，我的希望是分享实用的方法，帮助你开始用你的数据讲故事，以便你能做出更明智、更快、更有信心的决策，而不会迷失在数字中。

## 作为数据专业人士，你如何开始用数据讲故事？

### 从提出正确的问题开始

许多分析师将数据和处理视为一个技术元素，但它可以远远超出这一点。将数据视为你在决策过程中的关键合作伙伴。当你这样转变思维方式时，你开始将分析连接到行动。一个很好的开始方式是问自己：

+   我试图做出什么决策？

+   哪些数据可以帮助告知这个决策？

+   我错过了哪些洞察力，阻止我达到目标？

这种转变是将你的角色从报告信息转变为引导更明智决策的第一步。

### 关注正确的数据（而不是所有数据）

我们作为分析师需要避免的最大错误之一是陷入过多的数据中。更多的数字并不总是意味着更好的洞察力。你作为分析师的技能在于识别对你最有用的数据，过滤噪音，并专注于直接支持你目标的数据。

如何识别可操作的指标

+   定义你的目标：你试图实现什么？增长、效率、客户满意度？

+   使用领先指标：不要仅仅跟踪过去的绩效，寻找预示未来成功的信号。（例如，今天的客户参与度可以预测明天的留存率。）

+   简化：如果一个指标不能帮助你做出决策，那么它可能只是噪音。

例如，一份销售报告可能包含约200列和数千行，让你能够计算许多指标。但不要跟踪一切，更尖锐的问题来过滤你的数据将是：*哪些客户转换最快，哪些策略更快地关闭交易，或者什么模式推动收入？* 这就是行动所在。

这种方法帮助你基于洞察力采取行动，而不仅仅是观察数字。

### 在你的团队中建立一个数据驱动文化

作为分析师，你的角色是与数据工作，但如果你能鼓励你的团队也拥抱它，那岂不是很有趣！当你的团队中的每个人都理解用数据讲故事的价值时，决策可以变得更加有影响力。

*如何建立一个数据驱动团队*：

1.  鼓励数据讲故事：数字本身可能很枯燥，所以请让你的团队用背景、趋势和潜在行动来呈现洞察力，讲述故事，而不是统计数据。

1.  简化：创建非技术团队成员和利益相关者可以轻松理解的报告和仪表板。

1.  在会议中要求数据：当与团队讨论挑战或机遇时，问，“数据告诉我们什么？”

在每个层面都拥抱数据的劳动力将更加敏捷和高效，在决策方面。

### 超越数据，开始沟通影响

数据分析不应仅仅关于数字；它应该讲述一个推动行动的故事，并将分析转化为强大的叙事，以支持更明智的商业决策：

+   创建随时间讲述故事的决策循环：不要等待完美的数据，使用小的、迭代的循环来解析你的数据，并在过程中完善你的故事。用短期结果与长期趋势进行审查，并调整策略，以展示每个决策如何构建更大的商业叙事，从而为长期规划提供信息。

+   使用预测分析：预测模型将历史数据转化为面向未来的故事。例如，零售公司使用销售模式不仅来计划库存，而且来预测客户行为，制定与未来产生共鸣的活动。

+   利用人工智能和自动化：人工智能工具可以快速分析大量数据集，帮助你发现可能需要人类更长的时间才能发现的信息。自动化工具也让分析师能够专注于数字背后的“为什么”——制定出能够激发信心并推动变革的见解。如果你还没有探索过由人工智能驱动的分析，现在是时候了。

**总结，以下是你开始用数据讲故事的行动计划**：

要让你的见解真正与数据消费者产生共鸣，你需要超越分析师的常规思维，开始像讲故事的人一样思考，为此：

+   **确定商业目标**：明确识别需要决策的问题或机会。与利益相关者进行访谈，收集所有商业背景信息，这将指导你进行下一步的数据选择。

+   **定义你想要讲述的故事**：每个伟大的故事都始于一个目的。作为一名分析师，弄清楚你试图影响的关键问题或决策是什么，并确定你的故事将支持的商业时刻，以指导你的数据策略。

+   **确定塑造你叙述的数据**：确定你需要的数据集来讲述这个故事，因为并非所有数据都值得使用。你还需要确定哪些数字或趋势有助于解释正在发生的事情以及为什么？

    +   哪些差距可能会改变故事？

    +   你是否专注于*领先指标*来揭示即将发生的事情，或者只是关注历史数据？

+   **探索和可视化数据**：根据与目标的一致性、潜在风险和收益以及资源可用性，审查和评估所有可用资源（数据来源、仪表板、报告或分析工具），以揭示模式和主题。正确的数据分析可以帮助量化每个选项的潜在影响，从而允许进行更客观的选择。

+   **解释‘为什么’，而不仅仅是‘什么’**：数字是事实，所以它们可以告诉你*出了什么问题*，但故事解释*为什么它们很重要*。将你的发现转化为受众能够产生共鸣的见解。使用视觉、比较或现实世界的例子，使复杂的问题简单且易于记忆，并将抽象的指标转化为人们能够看到和感受到的东西。

+   **分享故事并跟踪其影响**：为你的数据故事制定一个清晰的社会化计划，以建立衡量有效性的指标。我强烈建议评估你故事的影响，了解它是如何被看待的，并在必要时进行调整。询问它激发的行动。

最好的领导者不仅收集数据，还利用这些数据做出明智、自信的决策。通过转变你的思维方式，关注正确的数字，建立数据驱动文化，并避免常见陷阱，你可以将原始数据转化为清晰的业务行动。

准备让数据为你工作了吗？从今天开始，做出一个决定，让讲故事引领方向。
