并非所有数据都是平等的：一些训练数据点对机器学习模型的训练影响远大于其他数据点。理解每个数据点的影响通常效率低下，并且往往依赖于模型的重复训练。*LossVal* 提出了一种新的方法，将数据估值过程有效地集成到人工神经网络的损失函数中。

## 什么是数据估值？

机器学习模型通常使用大型数据集进行训练。在大多数情况下，数据集中的所有训练样本对模型来说并不同等有帮助或具有信息性。例如，如果一个数据点是噪声的或标签错误，它对您的机器学习模型的信息性就较低。在我们的论文中的一个任务中，我们基于一些车辆参数，在一个 [车辆碰撞测试数据集上训练了一个机器学习模型，以预测碰撞对乘客的危害程度](https://link.springer.com/article/10.1007/s10618-021-00761-9)，其中一些数据点来自 80 年代和 90 年代的汽车！您可以想象，非常旧的汽车可能对模型预测现代汽车的重要性较低。

理解每个训练样本对机器学习模型影响的过程被称为 [数据估值](https://medium.com/towards-data-science/data-valuation-a-concise-overview-72e8cf12c755)，其中为每个训练样本分配一个重要性分数。数据估值是一个与数据市场、可解释人工智能、主动学习等领域相关的增长领域。已经提出了许多方法，如数据 Shapley、影响函数或 LAVA。要了解更多信息，您可以查看我最近发表的 [博客文章](https://medium.com/towards-data-science/data-valuation-a-concise-overview-72e8cf12c755)，其中介绍了不同的数据估值方法和应用。

## LossVal

LossVal 的基本思想是在训练模型的同时“学习”每个样本的重要性分数，类似于模型权重是如何学习的。这使我们免于多次重新运行模型的训练，并免于在训练过程中跟踪所有模型权重更新。

为了实现这一点，我们可以修改标准的损失函数，如均方误差（MSE）和交叉熵损失。我们将基于实例的权重纳入损失函数，并将其乘以一个加权距离函数。一般来说，LossVal 损失函数具有以下形式：

最优传输距离是将一个分布转换为另一个分布所需的最小努力。它也被称为地球迁移距离，来源于用一堆泥土填充洞的最快方式的类比。OT 可以定义为：

在更实际的意义上，通过改变权重来最小化 *OTᵥᵥ* 将会给与验证数据相似的训练数据点更高的权重。实际上，噪声样本会得到更小的权重。

## 实现

我们的实施方案和所有数据[可在 GitHub 上找到](https://github.com/twibiral/LossVal)。下面的代码展示了 LossVal 对均方误差的实现。

```py
def LossVal_mse(train_X: torch.Tensor, 
                train_y_true: torch.Tensor, train_y_pred: torch.Tensor,
                val_X: torch.Tensor, sample_ids: torch.Tensor
                weights: torch.Tensor, device: torch.device) -> torch.Tensor:
    weights = weights.index_select(0, sample_ids)  # Select the weights corresponding to the sample_ids

    # Step 1: Compute the weighted mse loss
    loss = torch.sum((train_y_true - train_y_pred) ** 2, dim=1)
    weighted_loss = torch.sum(weights @ loss)  # Loss is a vector, weights is a matrix

    # Step 2: Compute the Sinkhorn distance between the training and validation distributions
    sinkhorn_distance = SamplesLoss(loss="sinkhorn")
    dist_loss = sinkhorn_distance(weights, train_X, torch.ones(val_X.shape[0], requires_grad=True).to(device), val_X)

    # Step 3: Combine mse and Sinkhorn distance
    return weighted_loss * dist_loss**2
```

这个损失函数在 pytorch 中像任何其他损失函数一样工作，有一些特殊性：参数包括验证集、样本权重和批次中样本的索引。这是为了选择正确的权重来计算加权损失。请注意，此实现依赖于 pytorch 的自动梯度计算。这意味着样本权重向量需要成为模型参数的一部分。这样，权重的优化就能从优化器实现中受益，例如 Adam。或者，也可以手动更新权重，使用每个权重*i*的损失梯度。交叉熵的实现同样有效，但需要替换第 8 行。

## 结论

LossVal 背后的理念很简单：使用梯度下降为每个数据点找到一个最优权重。权重表示数据点的重要性。

我们的实验表明，LossVal 在 OpenDataVal 基准测试中实现了最先进的性能。LossVal 的时间复杂度低于我们测试的所有基于模型的方案，并且在面对不同类型的噪声和任务时表现出更稳健的性能。

总体而言，LossVal 为神经网络提供了其他最先进数据估值方法的一个高效替代方案。
