在几乎任何数字产品中，分析师经常面临构建数字客户档案的挑战——一组参数，以某种方式描述客户的状况和行为。

这些指标有哪些潜在的应用？

+   获取对用户行为的洞察

+   在机器学习模型中利用作为特征

+   制定个性化优惠的业务规则

一个简单的例子是电子商务，以下表格中列出了这些指标。

这些指标无处不在，但它们有什么问题呢？它们没有考虑到特定用户的记录或这个特定指标的动态。对于用户 1 来说，200 美元的花费算多吗？不清楚。然而，这种区别显著影响了我们接下来做出的商业决策。

即使在单个用户的背景下，200 美元对于业务来说，根据用户与产品的生命周期阶段的不同，可能有不同的含义。用户在用户引导、选择活动和重新激活期间花费的 200 美元是不同的。

那么，我们如何从对客户行为的数值描述过渡到更具有代表性的表示？例如，如何将“客户 7 天没有进行交易”的事实转化为对该特定客户是否异常或典型的个性化评估？挑战在于在不失去可解释性的同时，保持业务相关性，并避免黑盒模型。

一种简单的方法是分析指标的分布，并确定观察当前结果（即计算 p 值）的概率。这有助于我们了解该值与用户历史记录相比有多极端。

![正态分布，图片由作者提供](img/dc5358c2d99f483f05d344cdbd7cf3e7.png)

正态分布，图片由作者提供

但这里有什么挑战？在大多数情况下，指标的分布不是正态分布，这使得 p 值计算更加复杂。

随机指标可能具有与此类似的分布：

我们可以应用一些小技巧，将概率密度函数转换为累积分布函数（CDF）。在这种情况下计算 p 值要容易得多。

![CDF，图片由作者提供](img/d48e2cea0e4f99c3191a7ea23626f664.png)

CDF，图片由作者提供

因此，我们只需要从用户的指标中重建 CDF，这可以通过使用样条高效完成。让我们创建一个玩具示例。

* * *

想象你是一个电子商务平台，希望根据过去一周的用户活动来个性化你的电子邮件营销活动。如果一个用户与之前相比活动较少，你计划给他们发送折扣优惠。

你收集了用户统计数据，并注意到以下情况，针对名为约翰的用户：

+   约翰 15 天前第一次访问该平台。

+   在前 7 天（第 1-7 天），他访问了 9 次。

+   在接下来的 7 天（第 2-8 天），他访问了 8 次。

+   总共有 9 个值。

现在，你想要评估最近值与之前值相比有多极端。

```py
import numpy as np
visits = np.array([9, 8, 6, 5, 8, 6, 8, 7])
num_visits_last_week = 6
```

让我们创建这些值的 CDF。

```py
import numpy as np
import matplotlib.pyplot as plt

values = np.array(sorted(set(visits)))
counts = np.array([data.count(x) for x in values])
probabilities = counts / counts.sum()
cdf = np.cumsum(probabilities)

plt.scatter(values, cdf, color='black', linewidth=10)
```

![CDF，图片由作者提供](img/2ac814bc614acce7ae38ce6d9095d5c2.png)

CDF，图片由作者提供

现在我们需要根据这些值恢复函数。我们将使用样条插值。

```py
from scipy.interpolate import make_interp_spline

x_new = np.linspace(values.min(), values.max(), 300)  
spline = make_interp_spline(values, cdf, k=3)         
cdf_smooth = spline(x_new)

plt.plot(x_new, cdf_smooth, label='Сплайн CDF', color='black', linewidth=4)
plt.scatter(values, cdf, color='black', linewidth=10)
plt.scatter(values[-2:], cdf[-2:], color='#f95d5f', linewidth=10, zorder=5)
plt.show()
```

![带样条插值的 CDF，图片由作者提供](img/761f97fc2dbf01a1fe84ec9b1f616a98.png)

带样条插值的 CDF，图片由作者提供

还不错。但我们观察到红色点之间有一个小问题——CDF 必须是单调递增的。让我们用分段三次 Hermite 插值多项式来解决这个问题。

```py
from scipy.interpolate import PchipInterpolator

spline_monotonic = PchipInterpolator(values, cdf)
cdf_smooth = spline_monotonic(x_new)

plt.plot(x_new, cdf_smooth, color='black', linewidth=4)
plt.scatter(values, cdf, color='black', linewidth=10) 
plt.show()
```

![带分段三次 Hermite 插值的 CDF，图片由作者提供](img/d047391b805dffe5e9457dda81feb3ac.png)

带分段三次 Hermite 插值的 CDF，图片由作者提供

好的，现在它完美了。

为了计算我们当前观察到的 p 值（上周有 6 次访问）我们需要计算填充区域的面积。

![关键区域，图片由作者提供](img/66b351296a71e8afeb2979fe9607c2f7.png)

关键区域，图片由作者提供

要做到这一点，让我们创建一个简单的函数 `_calculate_p*value*`：

```py
def calculate_p_value(x):
    if x < values.min():
        return 0  
    elif x > values.max():
        return 1  
    else:
        return spline_monotonic(x)  

p_value = calculate_p_value(num_visits_last_week)
print(f"Probability of getting less than {num_visits_last_week} equals: {p_value}")
```

> 得到小于 6 的概率为：0.375

因此，概率相当高（我们可以将其与阈值 0.1 进行比较，例如），我们决定不向约翰发送折扣。我们需要为所有用户进行相同的计算。

在这篇文章中，我们看到了如何将原始用户指标转换为个性化评估，这可以导致更有效的商业决策。通过使用 CDF 和样条插值等统计方法，我们可以更好地理解用户行为背后的上下文，并提供基于数据的个性化、相关的优惠。
