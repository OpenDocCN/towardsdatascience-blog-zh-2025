## 优化问题的对偶性

每个优化问题都可以从两个不同的角度来审视。这个概念被称为优化问题的**对偶性**。例如，如果我们的原始目标是最大化公司在给定资源约束下的收入，那么同样的问题也可以通过最小化资源约束并试图销售一定数量的产品来表述。提供的原始线性优化问题被称为**原问题**。对于每个原问题，都有一个相应的**对偶问题**。如果原问题是最大化问题，那么对偶问题就是最小化问题，反之亦然。因此，对偶问题的对偶本身就是一个原问题。这种在优化问题特征描述中的基本对偶性被称为**对偶性**。

如上所述，目标函数的符号在原问题和对偶问题中正好相反。此外，原问题中使用的决策变量在对偶问题中被表述为约束，而原问题中的约束被表述为决策变量。这种对偶性在解决线性优化问题中具有多种优势。当求解器求解优化问题时，需要满足原问题和对偶问题的可行性条件。如果原问题有多个变量但约束较少，那么在这种情况下，对偶问题可能更容易解决。此外，对偶性提供了对优化问题结构的更深入洞察。

## 将原问题转换为对偶问题的步骤

将原问题转换为对偶问题的步骤可以总结如下。这受到了这个[视频](https://www.youtube.com/watch?v=SN67lGlID7M&ab_channel=StartPracticing)的启发。

1.  首先，将问题转换为标准形式。如果问题是最大化，则约束应采用小于等于的形式。如果问题是最小化，则约束应采用大于等于的形式。

1.  改变目标函数的符号。最大化问题变为最小化，反之亦然。

1.  原问题中的变量数将对偶问题中的约束数，反之亦然。

1.  原问题目标函数中的成本约束将对偶问题中约束的右侧常数，反之亦然。

1.  在构建对偶问题的约束条件时，使用原问题中约束矩阵的转置。

## 原问题到对偶问题的转换示例

**原问题**

假设我们想要最大化家具公司的收入。*x* 和 *y* 是特定时期椅子和桌子的销售数量，其单位销售价格分别为 90 和 75。

假设我们有关能效、劳动力数量和工作时间的资源约束。每把椅子需要3单位能量，每张桌子需要2单位能量，总体上，它们不能消耗超过66单位能量。每把椅子需要9单位劳动力，每张桌子需要4单位劳动力，总体上，它应该小于或等于180单位劳动力。每把椅子需要2个工作小时，每张桌子需要10个工作小时，总体上它应该小于或等于200个工作小时。这个问题以规范形式如下所示：

最大化 z = 90x + 75y

s.t. 3x+2y ≤ 66 (能量)

9x+4y ≤ 180 (劳动力)

2x+10y ≤ 200 (时间)

x, y ≥ 0

(椅子，桌子)

**对偶问题**

在上述问题中，我在原问题中有两个决策变量，三个约束条件。因此，对偶问题将有三个变量和两个约束条件。基于原问题中的每个约束条件，设*w1, w2*和*w3*为对偶问题中的三个变量。在这种情况下，*w1, w2, *和*w3*可以被认为是与能量、劳动力和时间约束相关的影子价格。**影子价格是指如果原问题的右侧约束条件改变一个单位，原问题目标函数的解将改变的量**。

我们得到：

min⁡ 66w1+180w2+200w3

s.t. 3w1+9w2+2w3 ≥ 90 (椅子)

2w1+4w2+10w3 ≥ 75 (桌子)

w1, w2, w3 ≥ 0

(能量，劳动力，时间)

## 将问题转换为标准形式

将原问题和对偶问题转换为**标准形式**意味着通过添加一些变量将不等式约束转换为等式。这对于确定最优性条件很有用，这将在下一节中进一步讨论。

在原问题中，我添加**松弛变量**将小于不等式约束转换为等式约束，如下所示。*h1, h2*, 和 *h3* 分别是能量、劳动力和时间约束的松弛变量。

最大化 z = 90x+75y

s.t. 3x+2y+h1 = 66 (能量)

9x+4y+h2 = 180 (劳动力)

2x+10y+h3 = 200 (时间)

x, y, h1, h2, h3 ≥ 0

在对偶问题中，我使用**剩余变量**将大于不等式约束转换为等式约束。*s1* 和 *s2* 是剩余变量。

min⁡ 66w1+180w2+200w3

s.t. 3w1+9w2+2w3-s1 = 90 (椅子)

2w1+4w2+10w3-s2 = 75 (桌子)

w1, w2, w3, s1, s2 ≥ 0

由于我们在标准形式中使用了等式约束，约束条件总是约束性的，这意味着我们正在使用所有可用的资源，而没有留下任何浪费。在这种情况下，在原问题的最优解中，松弛变量的值变为零。在对偶问题的最优解中，剩余变量的值也变为零。如果约束是非约束性的（例如，它有小于或等于约束），那么这些变量在最优解中的值可以是大于或等于零。

## 线性问题的最优性条件

对于一个线性优化问题，原问题的解和对偶问题的解只有在满足以下三个条件时才是最优的。（这受到了 Gurobi 优化这个视频[讲座](https://www.youtube.com/watch?v=y-B3ck1Fj44&list=PLHiHZENG6W8BeAfJfZ3myo5dsSQjEV5pJ&index=18&t=834s&ab_channel=GurobiOptimization)的启发。）

1.  **原问题可行性**

解需要是原问题可行的。这意味着原问题解中决策变量的值应该满足所有其约束条件。在我们的情况下，椅子和桌子（*x* 和 *y*）的数量应该分别位于能源、人力和时间的资源约束内。

3x+2y ≤ 66 (能源)

9x+4y ≤ 180 (人力)

2x+10y ≤ 200 (时间)

x, y ≥ 0

b. **对偶可行性**

这意味着对偶问题的解应该满足所有其约束条件。在我们的情况下，与能源、人力和时间相关的影子价格（*w1, w2,* 和 *w3*）应该满足椅子和桌子（成本系数）的价格约束。

3w1+9w2+2w3-s1 = 90 (椅子)

2w1+4w2+10w3-s2 = 75 (桌子)

w1, w2, w3, s1, s2 ≥ 0

c. **互补（或正交条件**）

假设 x^* 表示最优原问题的决策变量，h^* 表示同一原问题的松弛变量。假设 *s* 表示对偶问题中的松弛变量，*w* 表示对偶问题中相关的影子价格。在最优性条件下，原问题中的决策变量与对偶问题中相关的过剩变量之积应为零。这被称为**成本效率**，因为此条件确保我们正在最有效地利用我们的资源。

x^*s = 0

在我们的问题中，x * s1 = 0 和 y * s2 = 0 意味着：

x*(3w1+9w2+2w3-90) = 0

y*(2w1+4w2+10w3-75) = 0

此外，在最优性条件下，原问题中的松弛变量与对偶问题中的影子价格之积应等于零。这意味着我们正在资源效率上，没有资源浪费。

h*w = 0

h1*w1 = 0; h2*w2 = 0; h3*w3 = 0;

基本解，对偶间隙，强对偶性和弱对偶性

**基本解**

根据HiGHS文档中的这个很好的[定义](https://ergo-code.github.io/HiGHS/dev/terminology/)，如果一个线性问题在优化的方向上具有有界可行区域，并且是可行的，那么它在一个顶点处有一个最优解。“在这个顶点，决策变量可以分成与约束数量一样多的**基本变量**和**非基本变量**。在给定的解中，具有非零值的变量是基本变量，而具有零值的变量是非基本变量。这组解被称为**基本解**，而分割被称为**基**。

（如果使用单纯形算法解决线性问题，基本解和非基本解的集合[会随时间变化](https://math.stackexchange.com/questions/4249880/linear-programming-simplex-method-what-exactly-are-the-basic-and-non-basic-vari)。在迭代过程中，一些松弛变量将变为非基本变量，而其他变量将变为基本变量。单纯形法从一个基移动到另一个基，以改善目标函数，直到达到最优点，无法进一步改善目标函数。）

**对偶间隙**

最优原始值 *p^* 和最优对偶值 *d^* 之间的差异被称为对偶间隙 *p^*–*d^*。这个值总是大于或等于零。

**强对偶性**

如果原始目标值（解）和对偶目标值相同，则强对偶性成立，这意味着对偶间隙为零。强对偶性意味着问题通常是凸的，并满足某些约束条件。在这种情况下，对偶问题是原始问题的紧近似。

例如，如果一个家具工厂的椅子销售和桌子销售的最大收入等于资源使用的最低成本，包括能源、劳动力和小时数，这种条件被称为强对偶性。

**弱对偶性**

如果对偶间隙大于或等于零，则该条件被称为弱对偶性。这意味着对偶解小于或等于原始解。

在弱对偶性的情况下，对偶解更倾向于提供原始目标值的界限。如果原始问题是最小化问题，那么对偶可行解给出了原始目标值的下界。而如果原始问题是最大化问题，那么对偶可行解给出了原始目标值的上界。
